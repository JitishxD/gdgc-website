## About the Project
Google Developer Groups on Campus (GDGC) is an initiative of Google Developers. We at GDGC VIT Bhopal look forward to forming a community where we are able to convert our knowledge into real-time applications, help each student to develop in different fields of technology, and make use of our knowledge to build something that helps local businesses around us as well as their community.

## Tech Stack
The project is built using:
1. **Next.js**
2. **Tailwind CSS**
3. **Framer Motion**

## How to Contribute
**Fork and Branch:** Fork the repository and create a new branch for your changes.

## Setting up the project locally
To set up the project locally, follow these steps:

1. **Fork this repo**: Click the "Fork" button at the top right of this page.
2. **Clone the project**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/gdgc-website-2025-26.git
   cd gdgc-website-2025-26
   ```
3. **Install the dependencies**:
   ```bash
   pnpm install
   ```
4. **Run the project**:
   ```bash
   pnpm dev
   ```

## Pull Request Guidelines
- Ensure your code follows the existing style.
- Provide a clear description of the changes in your PR.
- Link the PR to the issue it addresses.
- Make sure the project builds successfully before submitting.
